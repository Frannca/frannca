Details
Develop a responsive and easy-to-use site

First site developed using Angular and API. It was the most challenging project, using new technologies for a large project. The API was built using Lumen, which was also used for the first time in a project. And the layout started to use the Semantic UI, in previous projects was used Bootstrap. This project did not use Angular CLI in the beginning, when it was updated from Angular 2 to Angular 4, it started to use the Angular CLI.
