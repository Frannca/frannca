Rebuilt the old site with modern technologies

Site developed with Angular to manipulate the data through the API, which was developed using Lumen. The client wanted a responsive layout, which was attended by using Semantic UI in conjunction with SASS for customizations. In addition to using Gulp for automation of tasks, and Git for versioning.
